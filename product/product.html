# 商品接口

# 商品搜索

  • 请求方式GET
  • 请求地址http:10.7.178.116:3000/api/search
  • 请求参数
    params:{
    name='商品名称',
}
  • 返回参数
    • 成功
    status:0,
message:'搜索成功',
data,
    • 失败
    status:1,
message:err,

# 商品列表

  • 请求方式GET
  • 请求地址http:10.7.178.116:3000/api/list
  • 请求参数（skip代表跳过x条数据，num代表查询y条数据）
    params:{
    skip=x,
    num=y,
}
  • 返回参数
    • 成功(skip表示下次开始跳过的查询数据)
    status:0,
message:'获取成功',
data,
skip=num,
    • 失败
    status:1,
message:err,

# 商品查询

  • 请求方式GET
  • 请求地址http:10.7.178.116:3000/api/find
